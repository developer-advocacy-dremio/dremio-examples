# Sample Prompts & GenAI

This directory contains resources for using Generative AI with Dremio, including prompt templates and agent skills.

## Files

- **[modeling_prompt_template.md](modeling_prompt_template.md)**: A universal prompt template to help LLMs understand Dremio's Medallion Architecture and generate SQL for Bronze, Silver, and Gold layers.
- **[example_prompt_application.sql](example_prompt_application.sql)**: A practical demonstration of the modeling prompt applied to a 3-table Commerce dataset.
- **[prompt_library.md](prompt_library.md)**: A collection of specialized prompts optimizing for Dremio's Text-to-SQL capabilities.
- **[external_agent_skills.md](external_agent_skills.md)**: Documentation and examples for configuring external agents (like Claude or Cursor) to interact with Dremio via Python or APIs.
