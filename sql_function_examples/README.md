# SQL Function Examples

This directory provides a reference for standard Dremio SQL functions across various categories.

## Files

- **[math_functions.sql](math_functions.sql)**: `ABS`, `CEIL`, `FLOOR`, `POWER`, `SQRT`, and other mathematical operations.
- **[string_functions.sql](string_functions.sql)**: `SUBSTR`, `CONCAT`, `REGEXP_LIKE`, `LOWER`, `UPPER`, and text manipulation.
- **[date_time_functions.sql](date_time_functions.sql)**: `DATE_ADD`, `DATE_TRUNC`, `EXTRACT`, `TO_TIMESTAMP`, and timezone handling.
- **[aggregate_functions.sql](aggregate_functions.sql)**: `SUM`, `AVG`, `COUNT`, `MIN`, `MAX`, and grouping sets.
- **[window_functions.sql](window_functions.sql)**: `RANK`, `DENSE_RANK`, `ROW_NUMBER`, `LEAD`, `LAG`, and frame specifications.
- **[conversion_functions.sql](conversion_functions.sql)**: `CAST`, `TO_CHAR`, `TO_NUMBER`, and type coercion.
- **[conditional_functions.sql](conditional_functions.sql)**: `CASE`, `COALESCE`, `NULLIF`, and logic branching.
- **[semi_structured_functions.sql](semi_structured_functions.sql)**: Handling JSON and complex types with `FLATTEN`.
