# SQL Admin Examples

This directory contains SQL scripts for Dremio administration and maintenance tasks.

## Files

- **[access_control_rbac.sql](access_control_rbac.sql)**: Demonstrates Role-Based Access Control (RBAC) setup, creating users, roles, and granting privileges on folders and datasets.
- **[reflection_management.sql](reflection_management.sql)**: Scripts for creating, refreshing, and optimizing Reflections (Dremio's query acceleration technology).
- **[data_maintenance.sql](data_maintenance.sql)**: Examples of table optimization (`OPTIMIZE TABLE`) and cleanup (`VACUUM`) for Iceberg tables.
- **[source_engine_management.sql](source_engine_management.sql)**: Scripts for configuring and managing Dremio execution engines and source connections.
