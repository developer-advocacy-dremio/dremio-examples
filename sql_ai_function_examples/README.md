# SQL AI Function Examples

This directory showcases Dremio's Generative AI functions for data enrichment and transformation.

## Files

- **[sentiment_classification.sql](sentiment_classification.sql)**: Using `AI_GENERATE_TEXT` to classify customer reviews or support tickets by sentiment (Positive/Negative/Neutral).
- **[data_transformation.sql](data_transformation.sql)**: Examples of using LLMs to clean messy data, extract entities, or reformat text strings directly within SQL queries.
