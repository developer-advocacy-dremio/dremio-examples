# Customer 360 Examples

This directory contains SQL scenarios focused on building unified customer views.

## Files

- **[retail_customer_360.sql](retail_customer_360.sql)**: A comprehensive scenario merging ERP, CRM, and Web Analytics data to create a "Golden Record" for retail customers. Includes Medallion Architecture (Bronze -> Silver -> Gold).
- **[hybrid_supply_chain.sql](hybrid_supply_chain.sql)**: (Note: Duplicate of the file in `hybrid_supply_chain/`) A control tower scenario merging data from Snowflake, S3, and Postgres.
